This guide helps you understand how to change or add new translations.

Qadam Flow uses Crowdin because it helps translators who don't know how to code. It also makes the approval process easier. The pipeline automatically syncs new text from the code and translations back into the code.

Target languages: **Russian, Uzbek, Kazakh, and English** — the rebrand is Central-Asia first.

## Contribute to existing translations

1. Create a Crowdin account.
2. Open the Qadam Flow Crowdin project (link will be added once the project is provisioned — open an issue on [GitHub](https://github.com/aiqadam/qadam-flow/issues) if you'd like access).
3. Click on the language you want to translate.
4. Click on "Translate All".
5. Select strings you want to translate and click "Save".

## Adding a new language

Open an issue on [GitHub](https://github.com/aiqadam/qadam-flow/issues) requesting the language — we will add it to the project and you can start translating.
