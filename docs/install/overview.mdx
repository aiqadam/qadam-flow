Qadam Flow can be self-hosted with **Docker** or **Docker Compose**.

<Tip>
Qadam Flow is **free** and **open source** — all features available to all users, no paywalls. See the [Roadmap](https://github.com/aiqadam/qadam-flow/blob/main/ROADMAP.md) for what's shipping next.
</Tip>

## Recommended Options

<CardGroup cols={2}>
  <Card title="Docker (Fastest)" icon="docker" color="#248fe0" href="./options/docker">

  </Card>

  <Card title="Docker Compose" icon="layer-group" color="#00FFFF" href="./options/docker-compose">
