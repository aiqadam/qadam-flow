If you would like to only show specific qadams to your embedding users, we recommend you do the following:

<Steps>
    <Step title="Tag Qadams">
        Tag the qadams you would like to show to your user by going to **Platform Admin -> Setup -> Qadams**, selecting the qadams you would like to tag and hit **Apply Tags**

        ![Bulk Tag](../resources/screenshots/tag-qadams.png)
    </Step>
    <Step title="Add Tags to Provision Token">
       You need to specify the tags of qadams in the token, check how to generate token in [provisioning users](./provision-users).

        You should specify the `qadams` claim like this:
        ```json
        {
            /// Other claims
            "piecesFilterType": "ALLOWED",
            "piecesTags": [ "free" ]
        }
        ```

        Each time the token is used by the embedding SDK, it will sync all qadams with these tags to the token's project. 
        The project will only contain the qadams that contain these tags.
    </Step>
</Steps>
