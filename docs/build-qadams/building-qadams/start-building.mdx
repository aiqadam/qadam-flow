This section guides you in creating a Gelato qadam, from setting up your development environment to contributing the qadam. By the end of this tutorial, you will have a qadam with an action that fetches a random ice cream flavor and a trigger that fetches newly created ice cream flavors.

<Info>
These are the next sections. In each step, we will do one small thing. This tutorial should take around 30 minutes.
</Info>

## Steps Overview 

<Steps>
  <Step title="Fork Repository" icon="code-branch">
    Fork the repository to create your own copy of the codebase.
  </Step>
  <Step title="Setup Development Environment" icon="code">
    Set up your development environment with the necessary tools and dependencies.
  </Step>
  <Step title="Create Qadam Definition" icon="gear">
    Define the structure and behavior of your Gelato qadam.
  </Step>
  <Step title="Add Qadam Authentication" icon="lock">
    Implement authentication mechanisms for your Gelato qadam.
  </Step>
  <Step title="Create Action" icon="ice-cream">
    Create an action that fetches a random ice cream flavor.
  </Step>
  <Step title="Create Trigger" icon="ice-cream">
    Create a trigger that fetches newly created ice cream flavors.
  </Step>
  <Step title="Sharing Qadams" icon="share">
    Share your Gelato qadam with others.
  </Step>
</Steps>
