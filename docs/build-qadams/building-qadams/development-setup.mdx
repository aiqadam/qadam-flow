## Prerequisites

- Node.js v18+
- npm v9+

## Instructions

1. Setup the environment

```bash
node tools/setup-dev.js
```

2. Start the environment

This command will start activepieces with sqlite3 and in memory queue.

```bash
npm start
```

<Note>
  By default, the development setup only builds specific qadams.Open the file
  `.env.dev` and add comma-separated list of qadams to make
  available.

For more details, check out the [Qadam Development](/build-qadams/building-qadams/development-setup#qadams-development) section.

</Note>

3. Go to **_localhost:4200_** on your web browser and sign in with these details:

Email: `dev@ap.com`
Password: `12345678`


## Qadams Development

When [`AP_QADAMS_SYNC_MODE`](https://github.com/activepieces/activepieces/blob/main/.env.dev#L14) is set to `OFFICIAL_AUTO`, all qadams are automatically loaded from the cloud API and synced to the database on first launch. This process may take a few seconds to several minutes depending on your internet connection.

For local development, qadams are loaded from your local `dist` folder instead of the database. To enable this, set the [`AP_DEV_QADAMS`](https://github.com/activepieces/activepieces/blob/main/.env.dev#L5) environment variable with a comma-separated list of qadams. For example, to develop with `google-sheets` and `cal-com`:

```sh
AP_DEV_QADAMS=google-sheets,cal-com npm start
```
