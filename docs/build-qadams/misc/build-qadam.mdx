You can use the CLI to build custom qadams for the platform. This process compiles the qadams and exports them as a `.tgz` packed archive.

### How It Works

The CLI scans the `packages/qadams/` directory for the specified qadam. It checks the **name** in the `package.json` file. If the qadam is found, it builds and packages it into a `.tgz` archive.

### Usage

To build a qadam, follow these steps:

1. Ensure you have the CLI installed by cloning the repository.
2. Run the following command:

```bash
npm run build-qadam
```

You will be prompted to enter the name of the qadam you want to build. For example:

```bash
? Enter the qadam folder name : google-drive
```

The CLI will build the qadam and you will be given the path to the archive. For example:

```bash
Qadam 'google-drive' built and packed successfully at packages/qadams/community/google-drive/dist
```

You may also build the qadam non-interactively by passing the qadam name as an argument.  For example:
```bash
npm run build-qadam google-drive
```
